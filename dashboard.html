ℹ️ How It Works

What this gateway does:

  • ✅ Allow - Safe prompts pass through
  • ❌ Block - Injection attacks are rejected
  • 🟢 Mask - Personal info is hidden (e.g., 0300-*******)

Detects:

  • 🔍 Prompt Injection & Jailbreak attacks
  • 📱 Phone numbers (Pakistan format: 03XX-XXXXXXX)
  • 📧 Email addresses
  • 🎓 Student IDs (STU-XXXXXX format)
  • 🔐 API Keys & Authentication tokens
  • 🔗 Composite PII patterns

Threshold Guide:

  • 📊 0.00 - 0.50: Strict (Maximum Security)
  • 📊 0.51 - 1.00: Balanced (Recommended)
  • 📊 1.01 - 2.00: Lenient (Fewer False Positives)
